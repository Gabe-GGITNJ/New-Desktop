# New-Desktop
Windows Desktop Build, Future Hackintosh

## Overview

It's been a while since I built my own desktop workstation.  The last barebones machine I assembled for myself was during the first Obama administration and it lasted a while. Before that I used a customized store bought machine. Buying and swapping out parts as needed. 
Since then I've been using laptops. 

With the announcement that Windows 10 was approaching End of Life, I decided that it was time to put my skills to the test once again. 
So last year I started the process.  I'm happy to say it's been a year and the MasterChief is still running smooth!

## Requirements/Features
* must be able to support Windows 11 Pro
* Motherboard must support TPM (Trusted Platform Module)
* Ability to run a stable standalone MacOS in the future
* Watercooling (optional) but nice to have in the near future

## Installation/Configuration
With the announcement of tariffs I  knew I had to get started sooner rather then later. I scoped out a few machines  that were ready to go out the box but they were outside of my budget. Similar to what I've done in the past I compared prices for the parts I needed and went to work. 

I went with a mATX case along with an AMD Motherboard.
